Ejercicio 4: Problema Dual de Bibliotecarios

Perspectiva Complementaria: Valoración de Períodos

Valor Óptimo: 26 Unidades (Igual al Primal)

Introducción a la Dualidad en Optimización

El Ejercicio 4 plantea el problema dual del Ejercicio 3. En programación lineal, todo problema primal tiene un problema dual asociado que proporciona una perspectiva diferente pero complementaria.

Problema PRIMAL (Ejercicio 3)

Pregunta: ¿Cuántos bibliotecarios necesito mínimamente para cubrir la demanda?

Perspectiva: Decisión de cantidad

Variables: x_i = cantidad de recursos a asignar

Objetivo: Minimizar costo total

Solución: 26 bibliotecarios

Problema DUAL (Ejercicio 4)

Pregunta: ¿Cuál es el valor marginal de satisfacer demanda en cada período?

Perspectiva: Valoración de restricciones

Variables: y_j = precios sombra (valores duales)

Objetivo: Maximizar beneficio total

Solución: Valor total = 26

Concepto Clave: El problema dual es el "espejo" del primal. Mientras el primal decide cuánto producir/asignar, el dual valora cuánto vale cada período.

Formulación Matemática del Dual

Variables Duales

y_j = Precio sombra (valor marginal) del período j donde j = 1, 2, 3, 4, 5, 6 Interpretación: ¿Cuánto vale satisfacer 1 unidad de demanda adicional en el período j?

Función Objetivo Dual (MAXIMIZAR)

W = 3y₁ + 2y₂ + 10y₃ + 14y₄ + 8y₅ + 10y₆ Objetivo: Maximizar el valor total de la demanda

Restricciones Duales

Cada restricción dual corresponde a una variable primal. Garantiza que el valor dual nunca sea "demasiado grande":

Para x₁: y₆ + y₁ ≤ 1 (Período 6 + Período 1) Para x₂: y₁ + y₂ ≤ 1 (Período 1 + Período 2) Para x₃: y₂ + y₃ ≤ 1 (Período 2 + Período 3) Para x₄: y₃ + y₄ ≤ 1 (Período 3 + Período 4) Para x₅: y₄ + y₅ ≤ 1 (Período 4 + Período 5) Para x₆: y₅ + y₆ ≤ 1 (Período 5 + Período 6) y_j ≥ 0, ∀ j (no negatividad)

Razón: Un bibliotecario cubre EXACTAMENTE 2 períodos consecutivos, por eso el valor dual de esos 2 períodos no puede ser mayor que 1.

Holgura Complementaria (Condiciones de Optimalidad)

Las soluciones primal y dual están relacionadas por las condiciones de holgura complementaria:

  • Si x_i > 0 (variable primal positiva), entonces la restricción dual i debe ser activa (= 1)
  • Si restricción primal tiene holgura, entonces su variable dual es 0
  • Si y_j > 0 (variable dual positiva), entonces la restricción primal j debe ser activa

Teorema de Dualidad Fuerte

TEOREMA DE DUALIDAD FUERTE: Si un problema primal tiene solución óptima finita, entonces su problema dual también la tiene, y: Z*_primal = W*_dual En nuestro caso: Z*_primal = 26 bibliotecarios W*_dual = 3(0) + 2(1) + 10(0) + 14(1) + 8(0) + 10(1) = 0 + 2 + 0 + 14 + 0 + 10 = 26 ✓ Los valores son IGUALES → Dualidad Fuerte Verificada

Interpretación del Valor Dual

El valor total de 26 puede entenderse de dos formas complementarias:

Perspectiva Primal

Necesitamos contratar 26 bibliotecarios total para cubrir toda la demanda.

Perspectiva Dual

El valor total de satisfacer la demanda es 26 (considerando precios sombra).
