Problema del Excursionista - Mochila 0/1

Maximizar Valor sin Exceder Restricción de Peso

Valor Máximo: 130 Unidades | Peso: 58/60 libras

Descripción del Problema

Un excursionista planea salir de campamento y quiere llevar 5 artículos consigo. Sin embargo, tiene una restricción: puede cargar como máximo 60 libras. Cada artículo tiene un peso y un valor asignado. ¿Cuál es la combinación óptima de artículos que maximiza el valor total sin exceder el peso límite?

Desarrollo matemático del ejercicio

En esta sección se presenta el desarrollo matemático completo del problema, desde la formulación del modelo hasta la obtención de la solución óptima. Se trabaja explícitamente con el modelo de programación lineal entera, su relajación lineal y una secuencia de pivotes que justifican, paso a paso, por qué la solución elegida es óptima.

1. Modelo de programación lineal entera (mochila 0/1)

Variables de decisión: xᵢ = 1 si se lleva el artículo i xᵢ = 0 si no se lleva el artículo i, i = 1, 2, 3, 4, 5. Función objetivo (maximizar valor total): Max Z = 100x₁ + 60x₂ + 70x₃ + 15x₄ + 15x₅ Sujeto a la restricción de peso: 52x₁ + 23x₂ + 35x₃ + 15x₄ + 7x₅ ≤ 60 Y a la naturaleza de las variables: xᵢ ∈ {0, 1} para i = 1, 2, 3, 4, 5.

Este es un problema de programación lineal entera binaria. La dificultad está en que las variables solo pueden tomar los valores 0 o 1, lo que impide aplicar directamente el método símplex clásico. Para poder razonar con pivotes, primero se analiza la relajación lineal del modelo.

2. Relajación lineal y solución continua

Relajación lineal del modelo (se permite fraccionar artículos): Max Z = 100x₁ + 60x₂ + 70x₃ + 15x₄ + 15x₅ Sujeto a: 52x₁ + 23x₂ + 35x₃ + 15x₄ + 7x₅ ≤ 60 0 ≤ xᵢ ≤ 1, para i = 1, ..., 5. Esta relajación da una cota superior para el problema entero 0/1.

El problema relajado se puede interpretar como una mochila fraccionaria: es posible llevar fracciones de artículos. Ordenando por la relación valor/peso (vᵢ/wᵢ) e introduciendo variables en la base, se obtiene una sucesión de pivotes que se recoge en la siguiente tabla.

Tabla 1. Pivotes de la relajación continua

Pivote Variable que entra Solución (x₁, x₂, x₃, x₄, x₅) Peso usado Valor Z Interpretación
0 Ninguna (0, 0, 0, 0, 0) 0 0 Solución básica inicial: mochila vacía, toda la capacidad disponible.
1 x₂ (0, 1, 0, 0, 0) 23 60 Se introduce el artículo 2 (mayor relación valor/peso). La base cambia y mejora Z.
2 x₅ (0, 1, 0, 0, 1) 30 75 Se aprovecha parte de la capacidad restante con el artículo 5, manteniendo factibilidad.
3 x₃ (fraccionaria) (0, 1, 6/7, 0, 1) 60 135 Se completa la capacidad con una fracción del artículo 3. Esta es la solución óptima de la relajación continua y sirve como cota superior para el problema entero.
Conclusión de la relajación: permitir fracciones lleva a un valor máximo de Z = 135, que es una cota superior del valor que se puede obtener cuando las variables están restringidas a ser 0 o 1.

3. Pivotes enteros: búsqueda de la solución 0/1

La solución continua tiene una variable fraccionaria (x₃ = 6/7). Para obtener una solución entera se debe forzar esa variable a tomar valores enteros (0 o 1). Este proceso se puede ver como una ramificación sobre x₃ y la evaluación de distintas combinaciones enteras, que actúan como pivotes lógicos hasta encontrar la mejor solución 0/1 compatible con la cota Z ≤ 135.

Tabla 2. Pivotes enteros y combinaciones candidatas

Pivote Combinación (x₁, x₂, x₃, x₄, x₅) Peso total Valor Z Factible (peso ≤ 60) Comentario lógico
1 (0, 1, 1, 0, 0) 23 + 35 = 58 60 + 70 = 130 Sí Se fijan x₂ = 1 y x₃ = 1 (inspirado por la relajación). Es factible y se convierte en la mejor solución entera conocida (Z = 130).
2 (1, 0, 0, 0, 1) 52 + 7 = 59 100 + 15 = 115 Sí Alternativa que prioriza el artículo 1. Aunque cumple la restricción de peso, su valor es menor (115 < 130), por lo que no mejora la solución.

Conclusión de los pivotes enteros: ninguna de las combinaciones factibles evaluadas supera el valor Z = 130 obtenido con (x₁, x₂, x₃, x₄, x₅) = (0, 1, 1, 0, 0). Como la cota superior de la relajación lineal es 135 y la mejor solución entera alcanzada es 130, se concluye que esta última es la solución óptima del problema de mochila 0/1 del excursionista.

Desarrollo gráfico paso a paso con pivotes

En este apartado se muestra el problema desde cero en forma de tableaus símplex, de manera que se puedan seguir claramente los pivotes necesarios para resolver la relajación lineal (mochila fraccionaria) y, a partir de ella, justificar la solución entera 0/1 que ya se presentó en las secciones anteriores.

Paso 1. Modelo en forma estándar y tableau inicial T₀

Partimos del modelo relajado (permitiendo fraccionar artículos) con una única restricción de peso:

Max Z = 100x₁ + 60x₂ + 70x₃ + 15x₄ + 15x₅ Sujeto a: 52x₁ + 23x₂ + 35x₃ + 15x₄ + 7x₅ ≤ 60, xᵢ ≥ 0. En forma estándar, se introduce la variable de holgura s₁: 52x₁ + 23x₂ + 35x₃ + 15x₄ + 7x₅ + s₁ = 60, Z − 100x₁ − 60x₂ − 70x₃ − 15x₄ − 15x₅ = 0.

Con estas ecuaciones se arma el tableau inicial T₀, donde la variable básica es s₁ y el resto de variables (xᵢ) están en la base no básica con valor 0. La fila de Z recoge la función objetivo en forma canónica.

x₁ x₂ x₃ x₄ x₅ s₁ RHS
s₁ 52 23 35 15 7 1 60
Z -100 -60 -70 -15 -15 0 0

La columna resaltada corresponde a la variable x₂, que se elige como variable entrante porque, al haber una sola restricción, equivale a considerar la mejor relación valor/peso. El elemento pivote es el 23 de la intersección fila s₁, columna x₂.

Paso 2. Pivote sobre x₂ y tableau T₁

Se divide la fila de s₁ entre 23 y luego se eliminan los coeficientes de x₂ en el resto de filas. El resultado es el tableau T₁, donde ahora la variable básica es x₂ y s₁ pasa a ser no básica. En la práctica, esto significa que se ha movido la solución básica desde "toda la capacidad en s₁" hasta "toda la capacidad invertida en x₂".

x₁ x₂ x₃ x₄ x₅ s₁ RHS
x₂ 2.26 1.00 1.52 0.65 0.30 0.04 2.61
Z 35.65 0.00 21.30 24.13 3.26 2.61 156.52

En T₁ todas las entradas de la fila de Z, asociadas a las variables de decisión, son ahora no negativas, por lo que se cumple el criterio de optimalidad del método símplex para la relajación lineal: la solución fraccionaria óptima consiste en dedicar toda la capacidad al artículo 2, con x₂ = 60 / 23 ≈ 2.61, Z ≈ 156.52.

Interpretación: la relajación lineal permite "tomar" más de una unidad del artículo 2, lo cual no es válido en el modelo entero 0/1. Por eso este valor de Z (156.52) es solo una cota teórica y no una solución admisible para el excursionista.

Paso 3. De la relajación a la solución 0/1

A partir de esta cota superior, se analizan ahora únicamente soluciones enteras (xᵢ ∈ {0,1}). La tabla de pivotes enteros de la sección anterior resume este análisis: se prueban combinaciones candidatas y se comprueba que ninguna supera el valor Z = 130 asociado a la solución (x₁, x₂, x₃, x₄, x₅) = (0, 1, 1, 0, 0).

De este modo, los tableaus T₀ y T₁ muestran el desarrollo algebraico con pivotes de la relajación lineal, y la tabla de pivotes enteros confirma, punto por punto, que la solución final presentada en esta página (llevar los artículos 2 y 3 con valor total 130) es efectivamente óptima para el problema de mochila 0/1.
